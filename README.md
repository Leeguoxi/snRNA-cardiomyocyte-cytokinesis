Cardiomyocyte cytokinesis is a key process in heart development and regeneration.
This project aims to use snRNA-seq data to minimize the confounding effects of canonical cell-cycle regulators (e.g., Cdk1, Aurkb) and, 
based on distinct cardiomyocyte states at developmental time points E18.5 and P4–P6, 
identify cardiomyocytes that enter the cell cycle but follow two different fates: successful division or polyploidization/multinucleation.
